# @moggan1337 Portfolio

A stunning, modern portfolio website showcasing my work as a Software Engineer and Open Source Enthusiast.

## ✨ Features

- 🎨 Modern dark theme with glass-morphism effects
- 📊 Animated stats and contribution graph
- 🗂️ Project showcase with filtering by category
- 📱 Fully responsive design
- ⚡ Fast loading with Next.js
- 🎭 Smooth animations with Framer Motion
- 🛠️ Easy deployment to GitHub Pages

## 🚀 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 🚀 Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions.

1. Push to `main` branch
2. GitHub Actions builds and deploys automatically

Or manually trigger a deployment from the Actions tab.
